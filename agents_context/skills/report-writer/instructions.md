# Report Writer — Template

Save to `/jobs/<slug>/application_kit_summary.md`:

```markdown
# Application Kit — <Company / Role Label>

**Date:** <date>
**Job description source:** /jobs/<slug>/job_description.md

## Company Research
<2-4 sentence summary + top 2-3 talking points, with confidence score>

## Resume
Tailored resume saved to: /jobs/<slug>/resume.md
<1-2 sentence note on what was emphasized/reordered, and any honest gaps flagged>

## Cover Letter
Saved to: /jobs/<slug>/cover_letter.md

## Interview Prep
Saved to: /jobs/<slug>/interview_prep.md
<1-2 sentence note on question mix and confidence score>

## Next Steps
<1-3 concrete suggestions, e.g. "address the Kubernetes gap before the interview">
```

Keep it factual — every claim in the report must trace back to something the
agent actually produced in this run, not assumptions.
