# Resume Tailoring — Examples

## Example 1
**Job description snippet:** "...looking for an engineer with strong SQL and
experience owning data pipelines end-to-end..."

**Original bullet:**
> Worked on backend services and some data-related tasks.

**Tailored bullet:**
> Owned an end-to-end ETL pipeline (SQL, Airflow) feeding daily reporting for
> 3 internal teams, reducing manual data-pull requests by ~70%.

*(Only acceptable if the candidate's source resume actually mentions an ETL
pipeline, SQL, Airflow, and the 70% figure — otherwise keep the original
wording and note the gap instead.)*

## Example 2
**Job description snippet:** "...cross-functional collaboration with design
and product..."

**Original bullet:**
> Built the checkout flow for the e-commerce app.

**Tailored bullet:**
> Partnered with design and product to ship a redesigned checkout flow,
> cutting cart-abandonment by 12%.

*(Again — only if "partnered with design and product" and the 12% figure are
true to the source resume.)*

## Example 3 — Honest Gap
**Job description snippet:** "...Kubernetes and container orchestration
required..."

**Resume has no container/Kubernetes experience.**

**Correct handling:** Do not add a fabricated bullet. Instead, surface this
in the summary: "Note: this role emphasizes Kubernetes experience which
isn't in your resume — worth addressing directly in your cover letter or
interview if you have any exposure, even informal."
